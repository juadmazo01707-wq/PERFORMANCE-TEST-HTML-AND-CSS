# Clipboard - Landing Page

A complete replica of the Clipboard website with responsive design, developed with HTML5 and CSS3.

## Characteristics

- **Responsive Design**: Optimized for mobile devices, tablets and desktop
- **HTML5 Semantic**: Clear and accessible structure
- **Modern CSS**: Use Flexbox and CSS Grid for layouts
- **Smooth Animations**: Elegant transitions and hover effects
- **Commented Code**: Complete documentation in the code

## Responsive Breakpoints

- **Desktop**: 1024px
- **Tablet**: 768px
- **Mobile**: 480px

## Design Specifications

### Main Colors
- **Main Green**: `#28cb8b`
- **Dark Gray**: `#717171`
- **Light Gray**: `#89939e`

### Fonts
- **Headings**: Semi Bold, sizes 36/44, 28/36, 64/76
- **Body Text**: Regular, sizes 18/28, 16/24
- **Font Family**: Arial, sans-serif

## How to Use

1. **Clone the project**
  - General steps:
    - Open your terminal (Git Bash, CMD, PowerShell, etc.).
    - Navigate to the folder where you want the repository (cd path/to/your/folder).
    - Run the git clone command with the URL of the remote repository (HTTPS or SSH).
    - Ready! You now have a local copy to start working.
2. **Open** `index.html` in a web browser
3. **Does not require** local web server to function
4. Images must be in the `img/` folder

## Customization

To modify the styles:

1. **Colors**: Edit color variables in CSS
2. **Fonts**: Change `font-family` and `font-size`
3. **Spacing**: Modify `padding` and `margin`
4. **Layout**: Adjust `flexbox` and `grid` properties
